# Agex Ventures Website

A modern, premium single-page website for **Agex Ventures**, a company operating in **Civil Construction** and **Oil & Gas Services** based in Jaipur, India.

---

## 🚀 Project Overview

This project is a fully responsive, visually premium corporate website designed to represent Agex Ventures professionally. It highlights the company's services, expertise, goals, and contact information in a clean and structured format.

## ⚙️ How to Use

1. Download or clone the project
2. Place your logo file as `logo.png`
3. Open `index.html` in any browser

---

## 🎨 Customization

You can easily customize:

* Company name & content → Edit inside HTML
* Colors → Change CSS variables in `:root`
* Logo → Replace `logo.png`
* Contact details → Update in Contact section
